# 🌍 Country Explorer

A modern React application that allows users to explore countries around the world, view detailed information, and see bordering nations.

This project focuses on clean component architecture, state management, API integration, and responsive UI design.

---

## 🚀 Features

- 🌎 View all countries
- 🔍 Search for a specific country
- 🌐 Filter countries by region
- 📄 View detailed country information:
  - Capital
  - Region & Subregion
  - Top-Level Domain
  - Currencies
  - Languages
- 🏳️ Display country flag
- 🔗 See bordering countries
- 🌙 Dark / Light mode (persisted using localStorage)
- 📱 Fully responsive design
- ⚡ Loading & error handling states

---

## 🛠️ Built With

- **React (Hooks)**
- **CSS Modules**
- **REST Countries API**
- **JavaScript (ES6+)**

---

## 🏗️ Architecture Highlights

- Modular component structure
- Separated API layer
- Reusable UI components
- Centralized theme management
- Defensive rendering for async data
